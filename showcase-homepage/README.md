# Showcase Homepage

Homepage estática, elegante e responsiva criada para demonstrar uma presença digital premium associada a tecnologia, cibersegurança, IA, automação e arquitetura.

## Ficheiros

- `index.html` — página completa com HTML, CSS e JavaScript embutidos.

## Características

- Design visual escuro e premium.
- Glass effect, gradientes, cards e microinterações.
- Layout responsivo para desktop, tablet e mobile.
- Sem dependências externas.
- Preparada para publicação via GitHub Pages.
- Conteúdo sem dados sensíveis ou contactos diretos.

## Como visualizar

Abrir diretamente:

```text
showcase-homepage/index.html
```

Se o GitHub Pages estiver ativo neste repositório, a página deverá ficar acessível em formato semelhante a:

```text
https://pestoura.github.io/html-devops-lab/showcase-homepage/
```

## Próximos melhoramentos possíveis

- Separar CSS e JavaScript em ficheiros próprios.
- Criar uma versão com Vite, Astro ou Next.js.
- Adicionar secção de projetos reais.
- Adicionar modo claro/escuro.
- Integrar formulário de contacto seguro.
- Publicar como portfolio principal em `pestoura.github.io`.
